# Cartão de Visitas Digital - Desenvolvedor Web & UI/UX

## Objetivo

Criar um cartão de visitas digital interativo e moderno para desenvolvedores web e designers UI/UX, apresentando informações pessoais, contatos e portfólio.

## Tecnologias Utilizadas

* **HTML5**: Estrutura do cartão
* **CSS3**: Estilização, animações e responsividade
* **JavaScript**: Interatividade, carrossel de projetos, microinterações e alternância de temas
* **Formspree**: Envio de formulário de contato sem backend
* **API de QR Code**: Geração dinâmica de QR code para portfólio

## Funcionalidades

* Foto do profissional com hover animado
* Carrossel de projetos com transição automática
* QR Code dinâmico para portfólio
* Contatos com animações de hover (WhatsApp, Instagram, LinkedIn)
* Botões de baixar e compartilhar com animações
* Formulário de contato integrado
* Alternância dinâmica de temas: Dark Mode e Lux Mode
* Totalmente responsivo para mobile e desktop

## Destaques

* Design moderno, interativo e personalizável
* Microinterações nos links e botões
* Fácil manutenção e atualização de conteúdos

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Interaja com o cartão: explore o carrossel, alterne os temas e use os botões de contato e compartilhamento.
4. Para personalizar, altere os textos, imagens e links diretamente nos arquivos HTML e JavaScript.
